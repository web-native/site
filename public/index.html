Tools

Libraries, tools, and specifications that bridge major gaps on the web, starting at the platform level.

  • Learn More ⟶
  • CHTML

    A suite of short specifications and technologies that bring a component architecture to the HTML language itself.

    #ScopedHTML · #ScopedCSS · #ScopedJS · #HTMLPartials
  • PlayUI

    A resilient, jQuery-inspired DOM API and an intuitive UI manipulation library.

    #AsynchronousDOM · #UIPhysics
  • Observer Components

    Application components and client-side APIs implemented as live, observerive objects.

    #ObserverActions
  • Observer

    A general-purpose observability API for JavaScript. (JavaScript's new Object.observe().)

    #ObserverActions
  • JSEN

    A general-purpose library for parsing JavaScript expressions that execute in user-defined scopes. (JavaScript's micro runtime.)

    #scopedJS
  • Commons

    A thoughtful utilities library for JavaScript; used across all Web-Native modules.
